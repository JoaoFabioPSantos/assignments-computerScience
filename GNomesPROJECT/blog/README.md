
# Gnomes Project
- Feito por João Fabio Pereira dos Santos, 0104N

## Descrição
A ideia foi fazer um tipo de blog de gnomos, ideia bem esquisita, mas implementei algumas coisas bem simples, só para focar no layout e em como seria mexer nesta ideia de site.

### Páginas
- Início, com os posts e com um efeito de gnomos seguindo o mouse;
- Página Gnomed, com um gnomo clicável que fala algumas coisas em uma caixa de texto e que é possível "agarrar para que ele fique pendurado pelo mouse;
- Contato, uma página simples com contatos logicamente falsos, em que tem-se um telefone que leva para um link separado de um "meme" e juntamente de um cogumelo que aparece uma recomendação real de blog.
